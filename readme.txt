1. To use ClipboardScope for static analysis, run through command: java -jar ClipboardScope_jar.jar [apk_path] android.jar
2. To analyze the static analysis result, execute the scripts in getType.ipynb
3. Files in this repo:
	a) all-soot-flowdroid: Source code in Eclipse project
	b) ClipboardScope-jar.jar: Executable jar of ClipboardScope
	c) android.jar: Dependency of the analysis
	d) evaluation_result.zip: Execution results of apps from Google Play
	e) getType.ipynb: Analysis scripts for the static analysis results
